# Binary Transformer Plugin: Image Resizer

A Binary Transformer Plugin to resize image

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bt_image_resizer'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bt_image_resizer

## Development
